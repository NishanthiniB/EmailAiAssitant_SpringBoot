📧 Smart Email Assistant

AI-powered email reply assistant built using Spring Boot, Spring AI, React, and Google Gemini API.

Generates smart, context-aware email replies directly inside Gmail/Outlook using a browser extension.

🚀 Features
✨ AI-generated email replies
📩 Gmail/Outlook integration
⚡ Real-time smart suggestions
🎯 Context-aware responses
🎨 Custom reply tones
🌐 Browser extension support
🔗 REST API integration
🔄 Application Flow
```text
Email UI
   ↓
Browser Extension
   ↓
React Frontend
   ↓
Spring Boot Backend
   ↓
Spring AI + Gemini
   ↓
Smart AI Reply
```
